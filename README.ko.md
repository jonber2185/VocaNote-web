# VocaNote

> 나만의 단어장을 만들고 플래시카드, 퀴즈, 예문 연습으로 단어를 완벽하게 익히세요.

![홈](./images/home.png)

[English](README.md)

---

## 기능

- **인증** — 회원가입 및 로그인으로 나만의 단어장 관리
- **단어장** — 단어장 생성, 검색, 탐색
- **단어 추가** — 단어와 뜻 추가, AI 자동 분석 지원
- **학습 모드**
  - 플래시카드 (`study/card`)
  - 객관식 (`study/multiple`)
  - 쓰기 연습 (`study/write`)
- **검색** — 단어장 검색

## 시작하기

### 사전 요구사항

- Node.js
- npm

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```
