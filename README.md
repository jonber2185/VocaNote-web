# VocaNote

> Build your own vocabulary sets and master words through flashcards, quizzes, and writing practice.

![Home](./images/home.png)

[한국어](README.ko.md)

---

## Features

- **Authentication** — sign up and sign in to manage your own word sets
- **Word sets** — create, search, and browse vocabulary sets
- **Add words** — add words and definitions, with AI-powered analysis
- **Study modes**
  - Flashcards (`study/card`)
  - Multiple choice (`study/multiple`)
  - Writing practice (`study/write`)
- **Search** — find existing word sets

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment
