# VocaNote

VocaNote is a web app for creating, organizing, and studying vocabulary word sets. Build your own word sets, add words with definitions, and study them through flashcards, multiple-choice quizzes, or writing practice.

[한국어](README.ko.md)

## Features

- **Authentication** — sign up and sign in to manage your own word sets
- **Word sets** — create, search, and browse vocabulary sets
- **Add words** — add words and definitions to a set
- **Study modes**
  - Flashcards (`study/card`)
  - Multiple choice (`study/multiple`)
  - Writing practice (`study/write`)
- **Search** — find existing word sets

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
